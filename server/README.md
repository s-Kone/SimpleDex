Install packages
```
npm install
```

Run server debug (dev) mode to enable console logging and automatic relaunch of server upon saving any file in the project
```
npm run serverstart
```
or
```
DEBUG=express-locallibrary-tutorial:* nodemon ./bin/www
```

Run server production mode
```
npm start
```
or
```
node ./bin/www
```
